# MDX와 Next.js로 블로그 만들기

이 글에서는 Next.js와 MDX를 사용하여 개인 블로그를 구축하는 방법에 대해 알아보겠습니다.

## 왜 MDX인가?

MDX는 마크다운에 JSX 컴포넌트를 추가할 수 있는 형식입니다. 이를 통해:

- 📝 마크다운의 간결함 유지
- ⚛️ React 컴포넌트의 상호작용성 추가
- 🎨 커스텀 스타일링 자유도

## 구현 과정

### 1. 프로젝트 설정

```bash
npx create-next-app my-blog --typescript --app
cd my-blog
npm install @next/mdx gray-matter
```

### 2. MDX 설정

next.config.js에서 MDX를 활성화합니다.

### 3. 글 작성

content 폴더에 카테고리별로 글을 작성합니다.

## 결론

MDX를 사용하면 정적 사이트의 성능과 동적 컴포넌트의 장점을 모두 얻을 수 있습니다.
